#### MODULE NAME
       ZE16_32

#### DEPENDENCIES
       SameBit

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     16     

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     32     

#### DESCRIPTION
       Takes  an  16-bit  input A, and outputs the A as a 
       zero-extended 32-bit value into Y.
